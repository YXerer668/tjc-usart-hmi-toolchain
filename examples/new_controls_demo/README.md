# New Controls Demo

`scene.json` is the offline clean-rebuild fixture for fixture-backed non-media
controls. `smoke.expect.json` is only the next COM36 smoke candidate: it checks
page `0` plus simple initial readback fields for the value/text-like controls.

This folder still does not claim live behavior for every object. Hotspot,
waveform drawing, touch behavior, and camera-only visual claims need separate
focused live evidence before they can be marked closed.
